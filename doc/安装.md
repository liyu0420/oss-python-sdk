# 安装 {#concept_32043_zh .concept}

本文介绍如何安装PYTHON SDK。

## 环境要求 {#section_q4s_p2f_lfb .section}

-   python版本：2.7以上
-   有紫光云账户，并开通OSS服务。

## 下载SDK {#section_kzv_nrg_4fb .section}

-   sdk基于aws boto3编写，需要安装boto3库

    安装步骤：

    ```
    pip install boto3
    ```

-   Github地址：[点击查看](https://github.com/liyu0420/oss-python-sdk)
